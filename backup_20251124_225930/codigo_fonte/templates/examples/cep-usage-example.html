Exemplo de Uso do CEP Automation

Este é um exemplo de como usar a função de busca automática de CEP em qualquer formulário. A função é carregada automaticamente e funciona com qualquer campo que tenha "cep" no ID.

Endereço Principal
Endereço Secundário
Como Funciona:
  • Digite um CEP válido em qualquer campo que tenha "cep" no ID
  • Quando você sair do campo (blur), a busca automática será executada
  • Os campos relacionados serão preenchidos automaticamente
  • Funciona com qualquer prefixo: PAR_cep, COL_cep, DIZ_cep, etc.
